# Робот-лимузин с интеллектуальным патрулированием

## Обзор проекта

Данный проект представляет собой симуляцию автономного робота-лимузина, оснащенного системой интеллектуального патрулирования. Робот выполняет циклические паттерны движения, имитируя патрулирование территории или обслуживание заданной области.

## Концепция робота

### Физические характеристики
- **Тип**: Дифференциальный привод (differential drive)
- **Форма**: Удлиненная конструкция, напоминающая лимузин
- **Управление**: Два независимых ведущих колеса
- **Стабилизация**: Пассивные опорные ролики
- **Модель**: Реализована в формате URDF/XACRO для Gazebo

## Алгоритм движения

Робот реализует двухэтапный циклический алгоритм патрулирования:

### 1. Движение к случайной точке
- Робот выбирает случайную цель в пределах заданного радиуса
- Использует пропорциональный контроллер для плавной навигации
- Автоматическая коррекция курса по мере движения
- Остановка с заданной точностью при достижении цели

### 2. Спиральное патрулирование
- После достижения случайной точки робот начинает спиральное движение
- Постепенное увеличение радиуса патрулирования
- Полное покрытие области вокруг точки остановки

### Параметры настройки
```yaml
